# Kaggle 野火竞赛项目指南

本文档包含了**代码运行指南**以及**详细分工与阶段任务**，方便团队成员随时查看进度和明确下一步工作。

---
### 1选. 已完成的内容
- **数据清洗：** `01_data_clean.ipynb` 已完成自动缺失值填补与去重，生成 `train_clean.csv` 和 `test_clean.csv`。
- **基线 Cox 生存模型：** `02_baseline_model.ipynb` 使用清洗后的训练数据训练并输出预测。
- **本地验证脚本：** `00_metric_validation.ipynb` 计算比赛混合指标：
  - `混合评分 = 0.3 * C-index + 0.7 * (1 - 加权 Brier)`
  - `加权 Brier = 0.3 * Brier@24 + 0.4 * Brier@48 + 0.3 * Brier@72`

### 2. 文件结构与说明
- `01_data_clean.ipynb`
  - 读取 `data_raw/` 数据，进行清洗并输出至 `data_clean/` 文件夹。
- `02_baseline_model.ipynb`
  - 使用 `data_clean/train_clean.csv` 训练 Cox 模型。
  - 输出本地验证预测：`data_clean/preds_baseline.csv`。
  - 输出测试集提交文件：`data_clean/Cox_Submission.csv`（可直接交 Kaggle）。
- `00_metric_validation.ipynb`
  - 读取 `train_clean.csv` 和预测结果，进行官方指标的本地打分计算。

### 3. 注意事项与常见问题查错
- 本地验证必须使用训练数据拆分的验证集，因为测试集没有目标标签。
- **关于 72 小时报错：** 训练集最长记录只到 66.99 小时，模型不认识超过此时长的数据。因此代码中**刻意加入了一个 72.0001 小时且未发生事件的假样本（伪样本）**，从而扩展模型的预测边界。如有报错，请检查假样本代码是否存在。
